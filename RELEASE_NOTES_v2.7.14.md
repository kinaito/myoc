# MyOC v2.7.14 发布说明

## 平台支持

- **macOS**：`.pkg` 安装包（双击安装到 /Applications）
- **Windows**：即将推出

## 更新内容

### 功能优化

- 应用内截图功能稳定，反馈时可附带当前界面截图
- 模型载入流程优化，提升稳定性
- 随机参数功能状态管理优化

### 体验修复

- 修复模型切换后部分设置未正确恢复的问题
- 修复 About 面板显示排版
- 优化下拉列表交互体验

### 安全与服务

- 线上服务安全策略升级
- 反馈系统稳定性提升

## 安装说明

### macOS

1. 下载 `Myoc-2.7.14.pkg`
2. 双击安装到 `/Applications`
3. 首次运行若提示"无法验证开发者"，请前往 **系统设置 → 隐私与安全性 → 仍要打开**

## 反馈

如遇问题，请使用应用内「💬 反馈」按钮提交。
