# MyOC - 快速创建原创的Live2D角色

MyOC 是一款为原创 Live2D 角色（OC，Original Character）设计的参数生成、随机与控制工具。它无需依赖复杂的后端服务器或 VPN，支持全本地端渲染与截图导出，并支持与 AI 智能体配合进行全自动化模型调试。

---

## ✨ 核心特性

* 🎨 **本地 Live2D 渲染**：支持导入 `.model3.json` 规范文件，实时在本地 OpenGL 画布中加载与流畅渲染模型。
* 🎲 **多维度参数随机与分类**：内置智能参数分类引擎，能够智能区分“外观参数”（头发、眼睛、服装颜色等）与“物理/动作参数”，支持按特定分类进行自由随机，快速寻找灵感。
* 📋 **随机方案保存与恢复**：允许将生成的满意随机参数保存为专属方案，支持随时重新载入与删除管理。
* 📸 **透明背景截图**：提供一键高保真截图功能。支持一键切换“透明背景”或“9种常用高纯度底色”，便于导出做立绘排版。
* 💬 **一键意见反馈**：内嵌无服务器化异常与反馈系统，使用安全签名（HMAC-SHA256）算法直接将问题描述及最新日志加密上报至开发者飞书群。
* 🤖 **AI 智能体对接**：开放本地双向 WebSocket 接口，允许智能体无缝驱动、调整和保存 Live2D 角色外观。

## 📥 下载安装

请前往 [GitHub Releases](https://github.com/您的用户名/myoc/releases) 下载对应平台的安装包。

### 🍎 macOS 安装指引 (Apple Silicon & Intel)
1. 下载 `Myoc.pkg` 并双击安装。
2. 安装完成后在应用程序（Applications）中找到 `Myoc.app`。
3. **首次运行安全解锁**：由于应用未进行商业签名，双击运行时可能会提示“无法验证开发者”。
   * **请按住 Control 键并右键点击** `Myoc` 应用图标，选择“打开”，在弹出的确认框中再次点击“打开”；
   * 或者是前往 macOS“系统设置 > 隐私与安全”，在底端找到 `已阻止打开“Myoc.app”`，点击“仍要打开”。

### 💻 Windows 安装指引
1. 下载并解压 `Myoc-Windows-x64.zip`。
2. 双击解压文件夹内的 `Myoc.exe` 启动。
3. **首次运行安全解锁**：如果 Windows Defender SmartScreen 拦截并显示“已保护你的电脑”，请点击警告框中的 **“更多信息”**，然后选择右下角出现的 **“仍要运行”**。

## 💻 开发者与技术架构说明

基于 Live2D 官方 SDK 封装的本地参数调试工具，支持 GUI 操作与 WebSocket API 远程控制。

**技术栈**: PySide6 · python-live2d (OpenGL) · asyncio + websockets

### 核心模块
* `src/core/api_server.py` — WebSocket API 服务器，端口 8765
* `src/core/self_diagnosis.py` — 自诊断引擎与根因分析
* `src/core/param_classifier.py` — 三层参数分类（规则库 / 智能体 / 手动）
* `src/core/randomizer.py` — 随机化引擎，支持局部 Presets 隔离
* `src/core/exp3_exporter.py` — 标准 `.exp3.json` 表情导出
* `src/core/model_loader.py` — 模型加载与状态持久化
* `src/ui/main_window.py` — 主窗口（含自定义 ExpressionDropdown 多选下拉）
* `src/ui/preview_widget.py` — OpenGL 实时预览

### 配置/输出目录
* 状态持久化目录: `settings/{model_name}/`
* 日志目录: `logs/`（`app.log`、`diagnosis.log`、`health.json`）
* API 端口: `8765`
* OpenGL 环境变量: `QT_OPENGL=desktop`
