# MyOC - 通用的 Live2D 模型捏脸应用

MyOC 是一款**通用的 Live2D 模型捏脸应用**，不依赖 VTube Studio（VTS）即可独立完成 Live2D 角色的形象、表情创作与管理。

---

## 🎯 产品定位

- **通用**：基于 Live2D 官方 SDK 的本地参数渲染，不绑定特定模型，不依赖 VTS 等第三方工具
- **完整工作流**：载入模型 → 捏脸 → 预览 → 保存表情文件，全链路闭环
- **多模创作**：支持同一工具的多个 Live2D 模型快速切换、分别捏脸
